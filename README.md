# covid_map
App contains:
Welcome screen, with buttons for login and signup.
Login Screen, Authentication enabled.
Signup Form, Authentication enabled, saved information to firestore.
Plugins used:
      -flutter fire
       -svg package
       -google maps sdk
       -geolocation sdk
